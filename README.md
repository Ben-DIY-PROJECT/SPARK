# SPARK Website (Germ 307)

This is a course website for Germ 307, showcasing German course content, updates, partner organizations, and contact information.

## Features

- **Modular Design**: Reusable modules in the HTML files allow easy replication and customization of sections like sliders, navigation, and content blocks.
- **Responsive Layout**: Optimized for various screen sizes with clean, modern styling.
- **Interactive Elements**: Includes JavaScript-powered sliders, tab switchers, keyboard navigation, and news-detail updates.
- **Lightweight**: No external dependencies, pure HTML/CSS/JS for fast loading.

## Structure

- `index.html`: Homepage
- `about.html`: About page with links to detail topics
- `partner.html`: Our Partners page
- `news.html`: News/events page
- `contact.html`: Contact information
- `about/`: Detail pages for about topics (`topic-a.html` to `topic-d.html`)
- `assets/css/`: Stylesheets (`base.css`, `layout.css`, `components.css`, and page CSS files)
- `assets/css/pages/partner.css`: Partner page styling
- `assets/js/main.js`: JavaScript for interactivity
- `media_doc/`: Media assets and placeholders
- `media_doc/Our Partners/`: Partner logos (`AATG.jpg`, `Goethe.jpg`, `NCSSMLogo.jpg`)

## Usage

1. Clone or download the repository.
2. Customize the HTML files with your own content.
3. Update images and logos in `media_doc/` as needed.
4. Open `index.html` in a browser to preview.

## Customization

- Edit CSS in `assets/css/` for styling changes.
- Modify JavaScript in `assets/js/main.js` for behavior adjustments.
- Replicate reusable modules (e.g., sliders, tabs) across pages by copying HTML structures.
- Update navigation links in all top-level pages when adding or renaming pages.
